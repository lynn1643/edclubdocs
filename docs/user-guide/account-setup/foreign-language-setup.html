Foreign language setup

TypingClub is a typing tool initially designed for English speakers. As it grew and gained popularity, users around the world began requesting more language options to accommodate the various keyboard layouts. Today, the program can be configured to provide a native experience for speakers of the following languages: Spanish, French, Portuguese, Italian, German, Slovak. To set up the student experience, follow the next steps:


  1. Install & Assign Lesson Plan

    Every class on TypingClub is automatically assigned our default lesson plan Typing Jungle (designed for typing in English). The lesson plans in other languages must be installed on your account and then assigned to your classes.

    • Go to Lesson Plans
    • Open the Lesson Plan Library
    • Locate the lesson plan of your choice and install it.

    2. Once the lesson plan is installed, assign it to your classes:

    • Go to Classes and open your class
    • Click the Lesson Plans link
    • Click Assign Lesson Plan to This Class and browse the list of installed lesson plans
    • Optional: Unassign Typing Jungle

  2. Update Keyboard Settings

    By default, students are able to choose their preferred keyboard layout. However, teachers have the option to select the layout for the entire class and thus lock the keyboard settings for the students. You might benefit from this option in a large-class setting as it ensures that everyone's account is pre-configured correctly.

    • Go to Classes and open your class
    • Click the Student Settings link
    • Lock Keyboard Language and Layout
    • Hit Update Settings to save

  3. Interface Language

    Finally, when students log into their accounts, they can update the language of their account in the top-right corner of their screen. This is the only setting that must be configured by the student. Updating the language of the interface is highly recommended as the entire communication of the program with the student will happen in the language selected.

Notice: At the moment, only the student interface is available in other languages. The teacher interface is currently still in English.
